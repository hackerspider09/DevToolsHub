# DevTools Hub

A collection of developer tools and utilities including formatters, linters, encoders, decoders, network calculators, cheatsheets, and software resource links.

All operations run entirely client-side in the browser.

---

## Tech Stack

- **Framework:** React 18 + Vite
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Icons:** Lucide React

---

## Getting Started

To run the project locally:

### 1. Clone the Repository
```bash
git clone https://github.com/hackerspider09/DevToolsHub.git
cd DevToolsHub
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173`.
